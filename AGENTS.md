# АРЧА — заметки для агента

## Проект

Одностраничный лендинг для кафе **АРЧА** (Токмок, Кыргызстан). Стек: **Vite 5 + React 18 + Tailwind CSS 3**. SPA без бэкенда — статика, разворачивается на GitHub Pages через Actions.

Слоган: «Кафе среди хвойного воздуха». Кухни: европейская, восточная, национальная, турецкая.

- Live: https://rjv-vi.github.io/archa/ (после первого пуша)
- Repo: https://github.com/rjv-vi/archa (планируемое имя)
- Источник стиля: `примеры/` (целевой — `photo_2026-05-11_19-42-52.jpg`, тёмно-зелёный bakery-вайб)
- Источник контента: `скрины/photo_2026-05-11_19-42-50.jpg` (инста `@archa.forest`)
- Папка `pictures for menu/` — куда складываются сгенерированные фото для меню; копируются в `public/images/menu/`. Сейчас там лежат фото от предыдущего проекта (La Cafe) — это намеренно, меню демо.

История: проект унаследован от шаблона la-cafe. Меню, картинки блюд и базовая структура секций оставлены как **демо для заказчика**, пока он не пришлёт реальный список блюд.

## Команды

```bash
pnpm install
pnpm dev       # http://localhost:5173 (base = '/')
pnpm build     # сборка в dist/  (base = '/archa/' для GitHub Pages)
pnpm preview
```

При проверке после правок: `pnpm build` должен пройти без ошибок и предупреждений. Пушить в `main` → автоматический деплой через `.github/workflows/deploy.yml`.

## Архитектура и стиль

Дизайн-направление — лесной/хвойный минимализм по мотивам логотипа АРЧИ: тёмно-зелёный фон, кремовая типографика, золотые акценты.

- **Палитра** (`tailwind.config.js`):
  - `forest` — основа Hero/Promo/CTA-блоков (`forest-800` `#1F2A1E` — главный тёмный; `forest-900` `#131C12` — более глубокий).
  - `sand` — кремовые тона (тон страницы, светлые секции). `sand-50` `#FBF7EC` (почти белый кремовый), `sand-100` `#F4ECD8` (фон страницы).
  - `gold` — золотой акцент (обводка лого, активные пилюли, золотые кнопки). `gold-400` `#D4A84B`, `gold-500` `#BB8E2E`.
  - `ink` — тёмный текст на светлых блоках, с зеленоватым подтоном.
  - **НЕТ** старых `coffee` / `rose` из la-cafe — заменены на `gold` / `forest`.

- **Шрифты** (`index.html`):
  - **Fraunces** (display, serif с характером — тёплый, лесной, ближе к bakery-референсу).
  - **Inter** (body).
  - **Caveat** (script-акценты).
  - **НЕ** Bricolage Grotesque (был в la-cafe — заменён на serif для лесного вайба).

- **Базовые классы** (`src/index.css`, `@layer components`): `.container-x`, `.display`, `.pill`, `.tap`, плюс утилиты `.safe-bottom`.

- **Анимации**: только хук `useInView` (`src/lib/useInView.js`) + компонент `Reveal` (`src/components/Reveal.jsx`) — тихий fade-up на скролл, plus CSS hover. **Не использовать framer-motion** — намеренно убрали для облегчения бандла.

## Секции (всего 4 — больше не добавлять без явного запроса)

1. **Hero** (`Hero.jsx`) — `АРЧА / ☕ / КАФЕ` через всю ширину на тёмно-зелёном фоне. Эмодзи-плейсхолдер по центру (фото продукта пока не используем — будет добавлено позже).
2. **Menu** (`Menu.jsx`) — пилюли категорий + грид белых карточек 2/3/4 колонки на кремовом фоне. Активная пилюля — `forest-800`. Данные в `src/data/menu.js` (демо от La Cafe, оставлено сознательно).
3. **Promo** (`Promo.jsx`) — тёмно-зелёный callout с золотыми акцентами, прямой аналог Drinko «New drink», но в палитре АРЧИ. Эмодзи `🍰` вместо фото.
4. **Footer** (`Footer.jsx`) — компактный с адресом (Токмок), телефоном, часами, WhatsApp и встроенной картой OSM (id=`visit` для якоря). Координаты: `42.856911, 75.283572`.

Логотип (`Logo.jsx`) — SVG-бейдж: тёмно-зелёный круг с двойной золотой обводкой, силуэт двух гор, надпись «АРЧА» сверху, тонкие засечки по бокам. По мотивам инсты `@archa.forest`.

Картинки блюд (`DishImage.jsx`) пробуют загрузить `/images/menu/{slug}.jpg`; на `onError` показывают плейсхолдер с эмодзи. Сейчас лежат демо-фото от La Cafe — заказчик пришлёт реальные.

Hero и Promo **специально не используют** `public/images/hero.png` / `promo.png` (они от La Cafe и не подходят по палитре) — показываем эмодзи. Старые файлы можно удалить после получения новых.

## Принципы при правках

- **Только русский** язык интерфейса.
- Цены в **сомах** (`«с»` после числа).
- **Не возвращать** удалённые секции (Marquee, Signature, Gallery, About-разворот, Visit-cinematic, MobileBar) без явного запроса — они были удалены сознательно как «перегруз».
- **Не возвращать** framer-motion и шрифты Playfair/Bricolage Grotesque — текущий дизайн осознанный (Fraunces для serif-вайба).
- **Не возвращать** палитру coffee/rose — заменены на gold/forest, это бренд АРЧИ.
- Никаких эмодзи в коде/комментариях (кроме данных меню — там они часть контента, и Hero/Promo-плейсхолдеров).
- Перед коммитом: `pnpm build` должен проходить.
- При смене названия репо — обновить `base` в `vite.config.js`.
